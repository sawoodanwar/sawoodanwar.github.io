## About This Site

This is a personal academic website built with HTML/CSS and hosted via GitHub Pages. It serves as the central hub for research outputs, publications, and professional information.

### Contents

- **About** — Research background, PhD details, and academic affiliation
- **Publications** — Journal articles and working papers with live citation metrics
- **Research Portfolio** — Links to reproducible code repositories
- **Contact** — Academic profiles (Google Scholar, ORCID, LinkedIn)
